# GoLibrary

Application to practice development using GoLang

## About

In this application the goal is to create a library system, allows the 
client to perform some operations described better in the section named `Features`

## Features

- Add new book
- Get all books
- Get specific book
- Filter books
- Delete book
- Update book

## Tech stack

- http
- dotenv
- docker
- mysql
- swagger
- logger
